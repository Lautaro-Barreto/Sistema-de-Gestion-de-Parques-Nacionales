# Sección de scripts de importación masiva de datos

Los datasets consultados son los siguientes:

* https://sib.gob.ar/areas-protegidas#buscador-general (archivo xslx que se puede leer como xml, trae los datos de los parques nacionales, regiones, provincias y tipos de parque)
* https://datos.gob.ar/dataset/turismo-parques-nacionales y https://datos.yvera.gob.ar/dataset/parques-nacionales/archivo/e8f7c0d7-02f9-4fdd-aa83-d61c13f565ae (csv con visitas por region por tipo de visitante)
* https://datos.yvera.gob.ar/dataset/organizaciones-distinguidas-sistema-argentino-calidad-turistica-sact/archivo/5c4c977c-d36c-4bda-b59f-448a17c9bc55 (csv con empresas concesionarias)
